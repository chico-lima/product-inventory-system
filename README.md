# product-inventory-system
Product Inventory System é um projeto educacional criado para consolidar conhecimentos em desenvolvimento backend e fullstack.
Atualmente, o sistema oferece uma API RESTful para controle de produtos em estoque, desenvolvida com Java e Spring Boot, utilizando Spring Data JPA e SQLite para persistência.

O projeto evolui de forma incremental, incorporando:

- Princípios REST;
- Boas práticas de organização de código;
- Testes unitários e de integração;
- Pipeline de CI/CD com GitHub Actions;
- Conteinerização com Docker;
- Integração com frontend em React.

O foco principal é sair da inércia técnica e construir um sistema real, do zero, simulando um ambiente próximo ao profissional.
